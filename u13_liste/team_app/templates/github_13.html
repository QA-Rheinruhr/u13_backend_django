GitHub Repository initialisieren

1. Account anlegen

Um ein Repository auf GitHub zu erstellen, musst du zunächst einen Account auf der GitHub-Website erstellen. Gehe dazu auf die GitHub-Website (https://github.com/) und folge den Anweisungen zur Erstellung eines Kontos.

GitHub Repository klonen

A. Repository mit Visual Studio Code klonen

Wenn du Visual Studio Code verwendest, kannst du ein GitHub-Repository direkt über die Benutzeroberfläche von VS Code klonen. Befolge diese Schritte:

  1. Öffne Visual Studio Code und gehe zum gewünschten Speicherort, an dem das Repository geklont werden soll.
  2. Gehe zum Menü "View" (Ansicht) und wähle "Command Palette" (Befehlspalette) oder verwende die Tastenkombination "Strg+Shift+P" (Windows/Linux) oder "Cmd+Shift+P" (Mac).
  3. Gib in der Befehlspalette "Git: Clone" (Git: Klonen) ein und wähle die entsprechende Option aus.
  4. Gib die URL des zu klonenden Repositorys ein. Diese findest du auf der GitHub-Website unter dem Button "Code" (Code).
  5. Wähle den gewünschten Speicherort aus und klicke auf "Clone" (Klonen).
  6. Visual Studio Code wird das Repository klonen und im ausgewählten Speicherort öffnen.

B. Repository über die GitHub-Website herunterladen

Wenn du das Repository ohne Verwendung von Visual Studio Code direkt über die GitHub-Website herunterladen möchtest, folge diesen Schritten:

  1. Rufe die GitHub-Website (https://github.com/) auf und gehe zum Repository, das du herunterladen möchtest.
  2. Klicke auf den grünen Button "Code" (Code).
  3. Wähle "Download ZIP" (ZIP herunterladen) aus der Dropdown-Liste.
  4. Das Repository wird als ZIP-Archiv heruntergeladen. Speichere es an dem gewünschten Speicherort auf deinem Computer.
  5. Extrahiere das ZIP-Archiv in einen Ordner, um das geklonte Repository zu erhalten.
