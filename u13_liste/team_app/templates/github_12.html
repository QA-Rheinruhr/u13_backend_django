GitHub

GitHub Repository initialisieren

1. Account anlegen

Um ein Repository auf GitHub zu erstellen, musst du zunächst einen Account auf der GitHub-Website erstellen. Gehe dazu auf die GitHub-Website (https://github.com/) und folge den Anweisungen zur Erstellung eines Kontos.

2.A. Repository per Web anlegen

Nachdem du einen GitHub-Account erstellt hast, kannst du ein Repository über die GitHub-Website anlegen. Folge diesen Schritten:

  1. Melde dich bei GitHub an und gehe zu deinem Dashboard.
  2. Klicke auf den grünen Button "New" (Neu) in der oberen rechten Ecke der Seite.
  3. Gib einen Namen für dein Repository ein und wähle optional eine Beschreibung aus.
  4. Wähle die Sichtbarkeit des Repositories (öffentlich oder privat).
  5. Optional: Wähle eine Lizenz, eine .gitignore-Vorlage oder füge ein README hinzu.
  6. Klicke auf den grünen Button "Create Repository" (Repository erstellen), um das Repository anzulegen.

2.B. Repository per VS Code anlegen

Wenn du Visual Studio Code verwendest, kannst du ein Repository direkt über die Benutzeroberfläche von VS Code erstellen. Befolge diese Schritte:

  1. Öffne Visual Studio Code und gehe zum gewünschten Speicherort, an dem das Repository erstellt werden soll.
  2. Gehe zum Menü "View" (Ansicht) und wähle "Command Palette" (Befehlspalette) oder verwende die Tastenkombination "Strg+Shift+P" (Windows/Linux) oder "Cmd+Shift+P" (Mac).
  3. Gib in der Befehlspalette "Git: Initialize Repository" (Git-Repository initialisieren) ein und wähle die entsprechende Option aus.
  4. Gib einen Namen für das Repository ein und wähle den gewünschten Speicherort aus.
  5. Optional: Erstelle eine .gitignore-Datei und wähle eine Vorlage aus, um Dateien und Verzeichnisse zu ignorieren.
  6. Bestätige die Initialisierung des Repositorys.
